Możliwe opcje¶

Przykład¶

Wyobraźmy sobie 2 zmienne pozostające ze sobą w pewnej zależności. Prosty wykres rozrzutu pozwoli nam zobrazować tą zależność.

$y = 2 + 0.5x - 0.1x^2 + 0.01x^3 + e$

x = np.random.uniform(0,10,200)
e = np.random.normal(0,1, 200)
def zaleznosc(exog):
    return 2 + 0.5*exog - 0.1*exog**2 + 0.01*exog**3
y = zaleznosc(x) + e

O czym mówi nam wykres rozrzutu?¶

Oczywiście taki wykres rozrzutu nie mówi nam nic o charakterze przyczynowym relacji pomiędzy tymi zmiennymi.

Zakładając jednak, że prawdziwe jest jedno z 2:

  • X --> Y
  • Y --> X

możemy oczekiwać pewnej asymetrii w rozkładzie obu zmiennych. Wynika to z faktu, że:

$y = f(x) + \epsilon$

gdzie

$\epsilon \sim N(0,\sigma)$

def probability(x,y,sd):
    mu = zaleznosc(y)
    dist = norm(loc = mu, scale = sd)
    return dist.pdf(x)

X = np.r_[0:10:100j]
Y = np.r_[0:10:100j]
xx, yy  = np.mgrid[0:10:100j, 0:10:100j]
Z = probability(xx,yy,1)

Proponowany algorytm¶

  1. Podziel zbiór danych X i Y na podzbiór treningowy i testowy
  2. Dopasuj modele: Y na X i X na Y na danych treningowych
  3. Przy użyciu funkcji obliczonych w 2 oblicz reszty regresji na danych testowych
  4. Oblicz wskaźniki zależności pomiędzy predyktorami a resztami: C(X,reszty od Y) i C(Y, reszty od X)
  5. Przyjmij, że model, w którym reszty są (w większym stopniu) niezależne od predyktorów - o niższym C - jest przykładem modelu z addytywnym szumem

    idx = np.random.randint(0,len(x)-1, int(len(x)/2))
    x_train = x[idx]
    x_test = x[-idx]
    y_train = y[idx]
    y_test = y[-idx]
    return x_train, x_test, y_train, y_test

  • HSIC - Hilbert-Schmidt Independence Criterion
