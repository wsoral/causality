Wyobramy sobie 2 zmienne pozostajce ze sob w pewnej zalenoci. Prosty wykres rozrzutu pozwoli nam zobrazowa t zaleno.

$y = 2 + 0.5x - 0.1x^2 + 0.01x^3 + e$

x = np.random.uniform(0,10,200)
e = np.random.normal(0,1, 200)
def zaleznosc(exog):
    return 2 + 0.5*exog - 0.1*exog**2 + 0.01*exog**3
y = zaleznosc(x) + e

Oczywicie taki wykres rozrzutu nie mwi nam nic o charakterze przyczynowym relacji pomidzy tymi zmiennymi. Zakadajc jednak, e prawdziwe jest jedno z 2:

  • X --> Y
  • Y --> X

moemy oczekiwa pewnej asymetrii w rozkadzie obu zmiennych. Wynika to z faktu, e:

$y = f(x) + \epsilon$

gdzie

$\epsilon \sim N(0,\sigma)$

def probability(x,y,sd):
    mu = zaleznosc(y)
    dist = norm(loc = mu, scale = sd)
    return dist.pdf(x)

X = np.r_[0:10:100j]
Y = np.r_[0:10:100j]
x, y  = np.mgrid[0:10:100j, 0:10:100j]
Z = probability(x,y,1)
